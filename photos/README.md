# Photos

Drop portrait photos in this folder. The dashboard picks them up automatically —
no code changes needed. If a photo is missing, the page shows a grey circle with
the person's initials instead.

## How to name each file

Use the person's name, all lowercase, spaces replaced with hyphens, then `.jpg`
(`.jpeg`, `.png`, `.webp` also work). Examples:

| Who                         | File name                       |
|-----------------------------|---------------------------------|
| Shepherd "Patra Amoako"     | `patra-amoako.jpg`              |
| Governor "Dorcas Longdon"   | `dorcas-longdon.jpg`            |
| Choir "True Love Choir"     | `true-love-choir.jpg`           |

Rules:
- lowercase everything
- replace each space with a single `-`
- drop apostrophes/brackets (e.g. "David Aseda Orleans-Lindsay" → `david-aseda-orleans-lindsay.jpg`)

Square photos look best (they're shown in a circle). ~400×400px is plenty.

Note: two shepherds with the exact same first name (e.g. two "Grace"s in different
hubs) will share one photo file. That's fine for now.
